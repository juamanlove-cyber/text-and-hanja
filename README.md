## 📖 프로젝트 소개

**"과거와 현재를 잇는 다리"** — 현대 뉴스 텍스트로 한자를 배우고, 역사 속 한자 텍스트를 현대어로 읽는 인터랙티브 한자 학습 플랫폼입니다.

한국어 어휘의 약 60–70%가 한자어로 구성되어 있으나, 디지털 시대에 한자 교육이 감소하면서 문해력 저하 현상이 심화되고 있습니다. 이 프로젝트는 전통적 한자 교육의 현대적 재해석을 통해 실용적 문해력 향상을 목표로 합니다.

## ✨ 주요 기능

### 📰 현대 텍스트 → 한자 학습
- **자유 텍스트 입력**: 뉴스 기사나 원하는 텍스트를 붙여넣으면 한자어를 자동 태깅
- **원클릭 한자 확인**: 한자어 클릭 시 한자, 뜻풀이, 어원 즉시 표시
- **전체 한자 변환 토글**: 스위치 하나로 모든 한자어를 한자로 일괄 전환
- **샘플 텍스트 제공**: 문해력 관련 샘플 텍스트 내장

### 📜 역사 텍스트 → 현대어 이해 (v3.0 신규)
- **4대 역사 문헌 수록**: 훈민정음 서문(1443), 난중일기(1592), 3.1 독립선언서(1919), 백범일지(1947)
- **원문 ↔ 현대어 모드 전환**: 한자 원문과 현대어 번역을 토글하며 비교 학습
- **현대어 풀이 표시**: 원문의 상세한 현대어 번역 제공
- **역사 텍스트 내 한자어 클릭 학습**: 현대어 모드에서도 한자어를 클릭하여 학습 가능

### 📝 단어장 & 퀴즈 시스템
- **개인 단어장**: 학습 중 한자어를 저장하고 관리 (localStorage 기반)
- **기본 퀴즈**: 한글 → 한자 4지선다 퀴즈
- **갑골문 퀴즈**: 갑골문 이모지와 어원 스토리를 활용한 퀴즈
- **학습 결과 피드백**: 정답률 및 점수 표시

### 🌍 다국어 지원
- **한국어 / 中文 / English**: 한자어 상세 정보를 3개 언어로 확인
- **중국어 병음 제공**: 각 한자어의 중국어 발음(拼音) 표시
- **네이버 한자사전 연동**: 심화 학습을 위한 외부 사전 바로가기

### 📱 모바일 최적화
- 스마트폰에서 앱처럼 사용 가능 (PWA 지원)
- 375×812 기준 모바일 퍼스트 디자인

## 🚀 사용 방법

### 웹에서 바로 체험

👉 **[여기를 클릭하여 앱 실행](https://html-doc-creator-juamanlove.replit.app)**

### 스마트폰에 앱처럼 설치

1. 위 링크를 모바일 브라우저에서 열기
2. 브라우저 메뉴 → **"홈 화면에 추가"** 선택
3. 바탕화면에 앱 아이콘 생성 완료!
