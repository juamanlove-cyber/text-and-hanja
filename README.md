## 📖 프로젝트 소개

한국어 어휘의 **60–70%는 한자어**입니다. 그런데 디지털 시대에 한자 교육은 점점 사라지고, 문해력 격차는 갈수록 벌어지고 있습니다.

**Text & Hanja 3.0**은 뉴스 기사를 붙여넣기만 하면 한자어를 자동으로 찾아주고, 클릭 한 번에 한자 · 뜻풀이 · 어원 · 중국어 발음 · 영어 번역까지 확인할 수 있습니다.

- **개발자:** 김주아 교수 (Prof. KIMJUA)
- **소속:** 국민대학교 중국인문사회연구소
- **연락처:** jua1225@naver.com
- **전문분야:** 디지털 인문학, 중국어·중국문화, 화교·화인 연구

## ✨ 주요 기능

### 📰 현대 텍스트 → 한자 학습

- 뉴스 기사나 텍스트를 **붙여넣기만 하면** 한자어 자동 태깅
- 클릭 한 번에 한자·뜻풀이·어원 표시
- 토글 스위치로 **모든 한자어를 한자로 일괄 전환**
- 샘플 텍스트 내장

### 📜 역사 텍스트 → 현대어 이해

- **4대 역사 문헌** 수록: 훈민정음 서문(1443) · 난중일기(1592) · 3.1 독립선언서(1919) · 백범일지(1947)
- **원문 ↔ 현대어** 모드 전환
- 현대어 속 한자어도 클릭 학습 가능

### 📝 단어장 & 퀴즈

- 학습 중 한자어를 **개인 단어장**에 저장
- **기본 퀴즈:** 한글 → 한자 4지선다
- **갑골문 퀴즈:** 어원 스토리 기반 퀴즈
- 정답률 및 점수 피드백

### 🌍 다국어 지원

- **한국어 / 中文 / English** 3개 언어 전환
- 중국어 병음(拼音) 제공
- **네이버 한자사전** 바로가기 연동
- 모바일 최적화 (PWA 지원)
