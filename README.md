---

## ✨ 주요 기능

- 🖱️ **원클릭 한자 확인**: 한자어 클릭 시 한자, 뜻풀이, 어원 즉시 표시
- 🔄 **전체 한자 변환**: 토글 스위치로 모든 한자어를 한자로 일괄 전환  
- 🔗 **네이버 사전 연동**: 심화 학습을 위한 외부 사전 자동 연결
- 📱 **모바일 최적화**: 스마트폰에서 완벽하게 작동
- 🎯 **맥락 학습**: 실제 텍스트 안에서 자연스러운 학습 경험

## 🚀 사용 방법

### 웹에서 바로 체험
👉 **[여기를 클릭하여 앱 실행](https://html-doc-creator-juamanlove.replit.app)**

### 스마트폰에 앱처럼 설치
1. 위 링크를 모바일 브라우저에서 열기
2. 브라우저 메뉴 → **"홈 화면에 추가"** 선택  
3. 바탕화면에 앱 아이콘 생성 완료!

## 📚 활용 분야

- **🏫 교육 기관**: 한자 교육 보조 도구로 활용
- **📰 언론사**: 뉴스 독해력 향상 서비스 제공

- **🔬 연구 도구**: 텍스트 한자어 분석 및 교육 효과 측정

## 📊 연구 배경

한국어 어휘의 약 60-70%가 한자어로 구성되어 있으나, 디지털 시대 한자 교육 감소로 인해 문해력 저하 현상이 심화되고 있습니다. 이 프로젝트는 전통적 한자 교육의 현대적 재해석을 통해 실용적 문해력 향상을 목표로 합니다.
